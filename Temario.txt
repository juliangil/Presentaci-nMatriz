	- Usar React puede traer ventajas y desventajas. Sus ventajas pueden ser: su modularizaciòn, alto rendimiento, transparencia, rapido
	  renderizado, uso de un DOM Virtual que para aplicaciones muy grandes mantiene su performance. 
	  Una desventaja puede llegar a ser el volumen de codigo o nùmero de archivos js generados, el flujo de datos va en una sola direcciòn
          (Acostumbrable). Incompatibilidades con otras librerìas, por ejemplo jQuery
	- Se aprovecha un proyecto ya hecho (Auth - Api - Angular y Node js) para la autenticacion de usuarios. Haciendo que matriz use los servicios 
	  de Api Auth, lo cual lo convierte en un sistema modular e integro.
	- El proceso de llenado o diligenciamiento de matriz de flexibilidad se automatiza, haciendo que dicha tarea sea amigable para los usuarios.
	- I+D como algunos otros desarrolladores de C&S tuvieron interacciòn con nuevas tecnologìas, lo cual fomenta capacitaciones y desarrollo de
